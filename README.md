# Machine-learning
A Machine Learning project implementing and comparing models — Linear Regression, Random Forest, Naive Bayes, KNN, and Decision Tree. 
Focused on evaluating performance to find the most accurate model using metrics like accuracy, precision, and F1-score.

🚀 Models Used
Linear Regression
Random Forest
Naive Bayes
KNN
Decision Tree

🧩 Tech Stack
Python
Scikit-learn
Pandas, NumPy
Matplotlib

📊 Goal

Compare and analyze model accuracy and performance on a given dataset.
